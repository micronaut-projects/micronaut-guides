== Lambda

Create a Lambda function. For the runtime, select `Custom Runtime`. Select the `x86_64` or `arm64` architecture to match the computer you use to build the GraalVM Native Image.

image::create-function-bootstrap.png[]

The Micronaut framework eases the deployment of your functions as a https://docs.aws.amazon.com/lambda/latest/dg/runtimes-custom.html[Custom AWS Lambda runtime].

The main API you interact with is https://micronaut-projects.github.io/micronaut-aws/latest/api/io/micronaut/function/aws/runtime/AbstractMicronautLambdaRuntime.html[AbstractMicronautLambdaRuntime]. This abstract class can be subclassed to create your custom runtime `mainClass`. That class includes the code to perform the
https://docs.aws.amazon.com/lambda/latest/dg/runtimes-custom.html#runtimes-custom-build[Processing Tasks] described in the Custom Runtime documentation.
