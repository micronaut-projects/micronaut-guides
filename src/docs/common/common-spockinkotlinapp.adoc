Micronaut is test framework agnostic. You can use https://spekframework.org[Spek], https://junit.org[JUnit] or https://spockframework.org/[Spock Framework].

In this Guide, we test the app with Spock.

We need to modify `build.gradle` to install Spock.

[source,groovy]
.build.gradle
----

include::{sourceDir}/build.gradle[tag=applygroovy]

dependencies {
...
..
include::{sourceDir}/build.gradle[tag=spock]
}
----

Edit `micronaut-cli.yml` to set Spock as the test framework:

[source,yaml]
.micronaut-cli.yml
----
include::{sourceDir}/micronaut-cli.yml[]
----
