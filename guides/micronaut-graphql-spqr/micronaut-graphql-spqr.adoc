== Getting Started

In this guide, you will create a Micronaut application written in @language@ that exposes a ToDo API with https://graphql.org/[GraphQL] and https://github.com/leangen/graphql-spqr[GraphQL SPQR].

SPQR is a code-first GraphQL library. Instead of describing the schema in a `.graphqls` file, you annotate Java types and service methods and let SPQR generate the schema.

Query the initial ToDo list:

[source,bash]
----
curl -X POST 'http://localhost:8080/graphql' \
     -H 'content-type: application/json' \
     --data-binary '{"query":"{ toDos { id title completed } }"}'
----

Create a new ToDo entry:

[source,bash]
----
curl -X POST 'http://localhost:8080/graphql' \
     -H 'content-type: application/json' \
     --data-binary '{"query":"mutation { createToDo(title:\"Write the SPQR guide\") { id title completed } }"}'
----
