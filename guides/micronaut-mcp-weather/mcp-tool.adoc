== MCP Tool

The MCP Server exposes two tools: `getWeatherForecastByLocation` and `getAlerts`.

source:Tools[]

callout:singleton[]
callout:constructor-di[arg0=WeatherClient]
callout:mcp-tool[]

`WeatherClient` is a Micronaut bean which uses the https://docs.micronaut.io/latest/guide/#httpClient[Micronaut HTTP Client] to consume
https://api.weather.gov API. Please, link:@sourceDir@.zip[download] the complete project to check the code.
