This repository contains tutorials for Micronaut. Tutorials are placed in the `guides` directory.
Tutorials can be written in multiple programming languages, including Java, Kotlin, and Groovy.

For a tutorial with the `slug` `micronaut-security-test`, the code is placed in the folder `guides/micronaut-security-test`.

For Java, the code is in `guides/micronaut-security-test/java`, for Kotlin in `guides/micronaut-security-test/kotlin`, and for Groovy in `guides/micronaut-security-test/groovy`.

Can you generate the Groovy code, if it does already exist, for the `micronaut-security-test` tutorial?

Moreover, can you generate the Kotlin code, if it already exists, for the `micronaut-security-test` tutorial?

It should be identical to the Java code, but the Groovy code and Kotlin code should be as idiomatic as possible.

For Groovy, instead of using JUnit 5, it should use the Spock Framework for testing. For Kotlin and Java, it should use JUnit 5.
